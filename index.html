🦊 Mengapa harus browser Firefox?

Google atau Google Chrome tidak mengizinkan website untuk mengirimkan notifikasi demi keamanan ponsel kamu. Oleh karena itu, saya menyarankan menggunakan aplikasi browser Firefox untuk mengaktifkan fitur notifikasi, karena sebagian fitur web ini mungkin tidak berfungsi tanpa izin dari sistem Google.

Website ini aman untuk digunakan — saya tidak menggunakan server online, sehingga tidak ada orang yang bisa masuk ke halaman website ini, termasuk developer sekalipun tidak bisa mengontrol kamu.
